# HelpHive

HelpHive connects hospitality teams with service providers for room attendant, public area attendant, and linen porter work.

## Live Links

- Web app: https://helphive.projects.himaiz.com
- Admin dashboard: https://admin.helphive.projects.himaiz.com
- Backend API: https://api.helphive.projects.himaiz.com
- Android APK: https://github.com/Helphive/helphive-app/releases/download/production-07/helphive-universal.apk
- Android release page: https://github.com/Helphive/helphive-app/releases/tag/production-07

## Repositories

- [helphive-app](https://github.com/Helphive/helphive-app) - React Native/Expo mobile app.
- [helphive-web](https://github.com/Helphive/helphive-web) - React web app for customers and providers.
- [helphive-admin](https://github.com/Helphive/helphive-admin) - Admin dashboard.
- [helphive-backend](https://github.com/Helphive/helphive-backend) - Node.js/Express API.

## Install Android APK

1. Open the Android APK link above on an Android device.
2. Download `helphive-universal.apk`.
3. Open the downloaded file.
4. Allow installs from the browser or file manager if Android asks.
5. Tap **Install**.

For support, contact support@helphive.projects.himaiz.com.
